# 📘 Assignment: Machine Learning in Python

## 🎯 Objective

In this assignment, you will build and evaluate a simple machine learning workflow in Python. You will practice preparing data, training a basic model, and interpreting model performance.

## 📝 Tasks

### 🛠️	Prepare and Explore the Dataset

#### Description
Load the provided sample data, separate features and labels, and inspect the dataset so you understand what the model will learn from.

#### Requirements
Completed program should:

- Load the sample dataset from a Python list or CSV-like structure.
- Split each row into input features (`x`) and target value (`y`).
- Print summary information such as total samples and a few example rows.


### 🛠️	Train and Evaluate a Simple Model

#### Description
Use the starter code to train a basic linear model (or gradient-descent approximation) and evaluate how well predictions match expected values.

#### Requirements
Completed program should:

- Train a model using the prepared features and target values.
- Generate predictions for the dataset (or a test split).
- Report at least one evaluation metric (for example, Mean Absolute Error) and briefly print what the result means.
